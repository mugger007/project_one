# Project One

A React Native mobile app built with Expo managed workflow.

## Features

- Navigation with React Navigation
- Styling with NativeWind (Tailwind CSS)
- State management with Zustand
- Geolocation with Expo Location
- Messaging UI with Gifted Chat
- Push notifications with Expo Notifications
- Gesture handling with react-native-gesture-handler

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm start`
3. Scan the QR code with Expo Go app on your device or run on simulator/emulator.

## Available Scripts

- `npm start` - Start the Expo development server
- `npm run android` - Run on Android emulator/device
- `npm run ios` - Run on iOS simulator (Mac only)
- `npm run web` - Run on web browser

## Troubleshooting

- Ensure Expo CLI is installed globally: `npm install -g @expo/cli`
- For iOS development without Mac, use Expo Go app.
- If NativeWind styles don't apply, restart the Metro bundler.
- Check Expo doctor: `npx expo-doctor` for dependency issues.
